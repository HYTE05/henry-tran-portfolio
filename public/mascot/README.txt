Phase 3 — mascot spritesheet (optional)

Add spritesheet.png here with:
- Exactly 5 frames in one horizontal row: idle, wave, point, excited, typing
- Equal width per frame (e.g. total width = 5 × frame width)
- Transparent PNG recommended; any square frame aspect works in the 96×96 viewport

The app probes /mascot/spritesheet.png on load. If missing or broken, the dashed placeholder is shown.
